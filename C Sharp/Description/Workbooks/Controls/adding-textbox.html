Adding TextBox - Aspose.Cells

This demo shows how to add textbox control in your worksheet using Aspose.Cells for .NET.

One way to stress important information on your report is to use a textbox. For example, you can enter text to highlight your company's name or to indicate the geographic region with the highest sales etc. Aspose.Cells provides TextBoxes class, which is used to add a new text box to the collection. There is another class TextBox, which represents a text box used to define all types of settings. The demo creates an excel file. Then by using simple Aspose.Cells APIs it adds two textboxes and apply different format setting to them. You may either open your resultant excel file into MS Excel or save directly to your disk.
