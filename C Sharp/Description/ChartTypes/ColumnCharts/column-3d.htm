Column 3D - Aspose.Cells

This online demo exhibits how to create a 3-D Column chart with simple, clustered and stacked flavors in a worksheet using Aspose.Cells for .NET.

This type of chart compares data points along two axes. Aspose.Cells component is a powerful component, which supports all the standard and custom charts to help you display data in more meaningful ways. The demo creates a workbook first and inputs some chart related data into the first two columns (A and B) of the first worksheet. The first column represents the category data (Region) and the second column represents values (Marketing Costs).

The demo creates a 3-D column chart representing marketing costs by region based on the different costs involving different regions. In the demo, you have been provided a sample snapshot of the chart and a few controls that represent the related list of chart data including a text box which represents chart type, five drop down lists which represent wall color, floor color, rotation angle, elevation angle and depth in percentage and a command button labeled Create Report to create and exercise the chart using your desired inputs. You can either open the resultant excel file into MS Excel or save directly to your disk to check the results.
