QuickStart - Aspose.Cells

These demos are quick start tutorials for Aspose.Cells.  After you check the source code, you will find how easy it is to use Aspose.Cells in your applications.

Note: To produce Excel spreadsheets, the machine to run Aspose.Cells doesn't need to have Microsoft Excel as Aspose.Cells itself is an Excel spreadsheet creation engine; To view the contents of Excel spreadsheets produced by Aspose.Cells , the machine to view them does need at least a Microsoft Excel Viewer. It can be downloaded for free.
