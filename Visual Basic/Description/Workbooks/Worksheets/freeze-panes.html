Freeze Panes - Aspose.Cells

This online demo describes how to freeze rows and columns (Freeze Panes) in a workbook using Aspose.Cells for .NET.

While manipulating larger worksheets you may require freeze panes in your worksheet. Aspose.Cells component provides users with the capabilities like freezing rows & columns at specified row / column index in the worksheet. Freezing panes helps you to view / work different areas of a larger worksheet at the same time. Freezing rows and columns allows you to select data that remains visible when scrolling in a worksheet. For example, keeping row and column labels visible as you scroll. The frozen rows are always the top rows while the frozen columns are always the left most columns. The demo creates an excel file and freezes the panes at C4 cell in the first worksheet in a workbook. You can either open the resultant excel file into MS Excel or save directly to your disk to check the freeze panes feature.
