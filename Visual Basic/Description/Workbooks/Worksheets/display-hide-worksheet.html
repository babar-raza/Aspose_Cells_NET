Hide/Unhide Worksheet - Aspose.Cells

This online demo describes how to Show / Hide worksheets in a workbook using Aspose.Cells for .NET.

Aspose.Cells component allows you to control the visibility of the worksheets in the excel files. The demo offers you two command buttons Hide and Display to exercise the tasks. When you click on Hide button, a workbook is created with a default worksheet. It makes the first worksheet invisible in the workbook. When you click on Display button, the demo creates an excel file makes the first worksheet visible in the workbook. You can either open the resultant excel files into MS Excel or save directly to your disk.
