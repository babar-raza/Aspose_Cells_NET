Setting Headers and Footers - Aspose.Cells

This online demo explains how to implement page setup settings related to headers and footers using Aspose.Cells for .NET.

Sometimes, you would like to configure page setup settings for your worksheets to control the printing process. For example, you may need to set Page Headers and Footers in your worksheet. These page setup settings are supported by Aspose.Cells component. The component allows you to add headers and footers to the worksheets at runtime. To add headers and footers at runtime, the component provides special APIs and some Script Commands to control the formatting of headers and footers. And you need to understand the vocabulary of the script commands. For reference, Please see Headers & Footers. The demo makes use of an existing excel file, opens it and implements date as a header and time as a footer in the first worksheet of the workbook. You can either open the output excel file into MS Excel or save directly to your disk and use Print Preview option to check the results in MS Excel.
