Data Validation - Aspose.Cells

This online demo explains how to add a List Object to the worksheet using Aspose.Cells for .NET.

Aspose.Cells allows you to create and maintain lists of various sorts for your need. The demo uses an excel spreadsheet file "ListObject.xls" and adds list object to the first worksheet in the workbook. You may either open your resultant excel file into MS Excel or save directly to your disk.
