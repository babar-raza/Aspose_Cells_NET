Data Sorting - Aspose.Cells

This online demo exhibits how to sort the data present in different cells of a worksheet using Aspose.Cells for .NET.

Sorting is performed on a list, which is defined as a contiguous group of data where the data is displayed in columns. Aspose.Cells allows you to sort Worksheet data alphabetically or numerically. It works in the same way as MS Excel does to sort data. The demo uses a template file "unsorted.xls" and sorts data for data range (A1:B14) in the first worksheet. You can either open the resultant excel files into MS Excel or save directly to your disk.
