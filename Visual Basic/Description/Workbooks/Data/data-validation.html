Data Validation - Aspose.Cells

This online demo explains how to implement data validation on different cells in a worksheet using Aspose.Cells for .NET.

Data validation allows you to build business rules into each cell so that grossly incorrect entries result in error messages. Business rules are the policies and procedures that governs how business operates. Aspose.Cells supports data validation based on your business rules and polices. The demo creates an excel spreadsheet file and applies data validation on B1 cell in the first worksheet which forces only number b/w 0 and 10 can be entered into the cell. An error message is shown in case of wrong entry. An input message is always displayed on the cell too. You may either open your resultant excel file into MS Excel or save directly to your disk.
