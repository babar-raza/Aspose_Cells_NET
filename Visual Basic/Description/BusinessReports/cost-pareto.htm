Cost Pareto - Aspose.Cells

This online demo illustrates how to create Cost Pareto business report using Aspose.Cells for .NET.

The Cost Pareto demo utilizes an xml file, reads the xml data from the file to generate two reports named Cost Data and Pareto Chart each representing a worksheet in the workbook. The first report calculates the annual cost of different cost centers while the second worksheet presents a column chart based on the first worksheet data. You can either open the resultant excel file into MS Excel or save directly to your disk to check the results.
